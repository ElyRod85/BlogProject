Front-End Design Portfolio Project

About this portfolio sample

By Eleonora Rodriguez

Share my portfolio project!

Hi! I'm Eleonora Rodriguez and I'm a Front-End developer and web designer.

The purpose of this portfolio sample is to emulate a WordPress-based blog, but using only HTML5, CSS3, JavaScript, and Bootstrap.

So why not just upload a WordPress-based sample and that's it? Well, mainly because GitHub doesn't support PHP in live deployments (for now), and also because the main point of my projects is for me to prove what I can do with HTML, CSS, and JS alone.

Since this website is only a portfolio sample to showcase my skills, most links will lead to either this explanatory blog post, my LinkedIn profile, or my GitHub portfolio.

Features:
  • As with all my portfolio projects, this blog sample is made by me from scratch, without using any templates.
  • These pages are built using flexbox, to ensure they're 100% responsive.
  • Social icons are from FontAwesome.
  • Google fonts used: Open Sans, Playfair Display, and Clicker Script.
  • All photos are royalty-free stock photos from Pixabay (except for my own photo in the sidebar, of course).
  • By the way, did you know I made my own free stock photo search app using the Pexels API? Click here to check it out!
